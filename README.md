# Homebrew Tap for Spool

[Spool](https://github.com/withakay/spool) is a spec-driven workflow management tool for AI-assisted development.

## Installation

```bash
brew tap withakay/spool
brew install spool
```

Or in a single command:

```bash
brew install withakay/spool/spool
```

## Development Installation (HEAD)

To install from the latest commit on main:

```bash
brew install --HEAD withakay/spool/spool
```

## Upgrading

```bash
brew update
brew upgrade spool
```
